웹 페이지를 구성하는 3 요소
			1. HTML : 웹 페이지의 구조와 내용(컨텐츠)
			2. CSS : 웹 페이지의 꾸밈 요소 (스타일 시트)
			1. JavaScript : 웹 페이지의 동적인 요소 구현

			HTML5 이전 W3C의 표준이 생기기 전에
			Active X 등의 비표준 기술이 많았다.
			웹 브라우저 마다 다른 호환성으로 문제가 있었음.
			MS silverlight, Adobe Flash 등 전부 비표준

			HTML5 모바일, PC, 태블릿, 브라우저 관계 없이 통일된 표준을 가진

			# 이미지 태그 img
			이미지 태그를 사용하면 HTML 문서에 이미지를 삽입할 수 있다.
			src는 이미지 태그에 필수로 작성해야 하는 속성이고 그 외 속성은 생략 가능
			src 속성에 지정할 수 있는 이미지의 타입은 BMP/GIF/PNG/JPG(JPEG)
			이미지 태그에 사용 가능한 속성:
			1. src : 이미지의 주소
			2. alt : 이미지 로딩에 실패하면 나타나게 할 텍스트
			3. width : 이미지의 너비(생략하면 원본 크기 출력)
			4. height : 이미지의 높이(생략하면 원본 크기 출력)

			# 하이퍼 링크 a
			웹 사이트는 여러 개의 HTML 페이지를 가지고 페이지들이 상호 연결하는 방식으로
			전체를 구성한다. HTML 페이지에서 다른 HTML 페이지로 연결하는 고리를 하이퍼링크 라고 함.
			하이퍼링크는 텍스트나 이미지에 다른 웹 주소를 달아서 만든다.
			하이퍼링크는 같은 사이트에 다른 HTML 페이지 뿐 아니라 다른 사이트에 있는 웹 페이지도
			연결 가능하다. 하이퍼링크를 따라서 다른 웹 페이지로 이동하는 것을 네비게이션이라고 부름
			혹은 링크라고 줄여서 표현.

			# 하이퍼링크 a 사용법
			1. a 태그는 여는 태그와 닫는 태그 모두 작성
			2. a 태그는 하이퍼링크 목적의 태그로 이동하려는 페이지의 주소를 필수 작성 
			3. href 속성에 URL 입력
