# DOM 객체를 JS코드로 식별하는 방법
        # document 객체의 메서드 / 컬렉션 / 프로퍼티
        
        # 프로퍼티
        location : 현재 문서의 URL 정보를 가진 객체
        readyState : 문서의 현재 로딩 상태
        title : 문서 제목
        head : 헤드 객체
        body : 바디 객체
        activeElement : 현재 포커스 된 요소

        # 컬렉션
        images : 문서 내 img 객체들의 컬렉션
        link : 문서 내 링크 객체들 컬렉션
        form : 문서 내 폼 객체 컬렉션 
        
        # 메서드
        # id를 제외하고는 전부 복수형(Elements)
        document.getElementById('a') : id
        document.getElementsByClassName() : 클래스명
        document.getElementsByName() : name 속성
        document.getElementsByTag() : 태그명
        addEventListener() : document 객체에 이벤트 리스너 추가
        close() : document 객체에 HTML 컨텐츠를 브라우저에 전부 출력하고 더이상 쓰기를 받지 않음.
        createElement() : HTML 태그의 동적 생성 - 현재 문서를 유지하면서 원하는 위치에 태그 추가
        removeEventListener() : 인벤트 리스너 제거
        write() : 쓴다.
        writeln() : 쓴다. (줄바꿈 된다)
